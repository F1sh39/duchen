# 此仓库是F1sh分支
## **DaoDao**(du)和**F1sh**(chen)的联合项目

## 项目分工  
该项目是Dao与F1sh于2021/9(大四)开设的电商mall平台项目，采用前后端分离方式搭建项目
+ Dao负责前端部分的编写
+ F1sh负责后端部分的编写

## 项目介绍  
仿照天猫商城，实现买家端的个人信息模块、首页模块、商品搜索模块、商品详情页模块、购物车模块、下单模块、订单模块等功能
+ 后端技术栈  
    SSM(Spring+SpringMVC+MyBatis) + MySQL + Tomcat
